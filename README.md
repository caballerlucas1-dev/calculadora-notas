**Proyectos Python Colección de 5 proyectos en Python desarrollados para la materia. Cada uno abarca un tema distinto y se ejecuta desde la terminal.**
1. Calculadora de Notas
Calcula el promedio escolar. Ingresás materias con sus notas y el programa te dice si aprobaste o desaprobaste cada una, mostrando el promedio general al final.

2. Gestor de Tareas
Aplicación para gestionar tareas pendientes. Podés agregar, completar y eliminar tareas. Todo se guarda en un archivo JSON para que persista entre sesiones.

3. Conversor de Unidades
Conversor de temperatura, distancia y peso. Soporta Celsius/Fahrenheit/Kelvin, kilómetros/millas y kilogramos/libras, entre otras combinaciones.

4. Adivina el Número
Juego donde la computadora elige un número al azar y el usuario tiene que adivinarlo. Tiene tres niveles de dificultad y muestra cuántos intentos tardaste.

5. Cifrado César
Implementación del cifrado César clásico. Permite cifrar y descifrar mensajes con una clave numérica, y también tiene modo fuerza bruta para probar todos los desplazamientos posibles.


Python 3.x
No requiere librerías externas
